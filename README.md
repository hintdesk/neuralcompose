# NeuralCompose Thunderbird Extension

This is a minimal template for a Thunderbird plugin (WebExtension) named **NeuralCompose**.

## Structure
- `manifest.json`: Extension manifest file.
- `background.js`: Main background script.

## Installation
1. Open Thunderbird.
2. Go to Add-ons > Extensions > Install Add-on From File...
3. Select the `NeuralCompose` folder (zipped).

## Development
- Edit `background.js` to add your extension logic.
- Update `manifest.json` as needed for permissions and features.
